# Mystery Hex Flower Investigations
## Part 3: GM Guidance

### 1. Case Construction

#### Basic Case Template
1. **Core Mystery Elements**
   - Central crime/event
   - Key suspects/witnesses
   - Critical locations
   - Timeline of events
   - Required evidence

2. **Zone Population**
   ```
   Red Zone (1-3):
   • Final revelation
   • Critical breakthrough
   • Case-solving evidence

   Blue Zone (4-7):
   • 2-3 physical clues
   • 2-3 key witnesses
   • 1-2 document trails
   • 1-2 hidden connections

   Green Zone (8-10):
   • Analysis opportunities
   • Pattern recognition
   • Evidence processing

   Yellow Zone (11-14):
   • Relevant complications
   • Meaningful obstacles
   • False leads

   Purple Zone (15-19):
   • Initial leads
   • Starting points
   • New directions
   ```

### 3. Running Investigations

#### Session Flow
1. **Opening**
   - Review current position
   - Check resource status
   - Note clock status
   - Set scene

2. **Scene Resolution**
   - Describe environment
   - Present opportunities
   - Handle resource checks
   - Navigate hex flower

3. **Progress Tracking**
   - Update resources
   - Record evidence
   - Adjust clock
   - Note position

#### Pacing Guidelines
```markdown
Early Game (Clock 0-2):
• Establish scene
• Gather initial evidence
• Build resource pool
• Develop leads

Mid Game (Clock 3-4):
• Connect evidence
• Face complications
• Manage resources
• Build tension

End Game (Clock 5-6):
• Resolve mysteries
• Overcome obstacles
• Make breakthroughs
• Reach conclusion
```
